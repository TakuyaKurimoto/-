・ファイル「nensyukeisan」の説明

→phpとhtmlだけで作ったシンプルな計算サイト。
→年収と生活費の各項目（家賃、交際費など）を埋めると、ひと月あたりの手取り額と生活費の合計を計算し、月いくら貯金できるか算出する。
